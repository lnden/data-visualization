## When to use

登录用户名、密码图标, 系统导航图标, 一些下载、导入、导出、搜索、清空等svg图标

## Method of use

```vue  
<svg-icon icon-class="icon-name" class="custom-style" />
```

## Parameter Settings

| 参数      | 说明                  | 类型   | 默认值 | 必填 | 版本 |
| --------- | --------------------  | ------ | ----- | ----- | ---- |
| iconClass | svg图标名称           | string | voild | true  |      |
| className | 自定义class名称       | string | voild | false |      |
